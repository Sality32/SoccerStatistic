# SoccerStatistic

How to run the project
- Clone the Project
- Copy the .env.example to your .env
- Update the credential database with your own database
- Run mysql
- Run in terminal
  > "composer install"
  
  > "php artisan key:generate"
  
  > "php artisan migrate"
  
  > "php artisan db:seed"
  
  > "php artisan serve"
